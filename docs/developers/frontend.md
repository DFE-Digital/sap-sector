# Frontend Development

---

## Stack

- GOV.UK Design System
- DfE Frontend
- SASS
- npm

---

## Asset rules

- wwwroot contains build output
- wwwroot/lib is generated
- SASS source lives outside wwwroot
- compiled CSS goes to wwwroot/css

---

## Accessibility

Accessibility is mandatory:
- keyboard navigation
- focus visibility
- clear error messages
